VIGENERE CIPHER
Vigenère cipher is a polyalphabetic substitution encrypting method that is also convenient to encrypt and decrypt manually. This means even if one loses access to the tools used during encryption, it is still possible to decrypt the content using good old pen and paper, or by writing a simple program using any computer language.
RULES
• Whitespace is omitted.
• Letter-in, letter-out.
• Number-in, number-out.
• Output case is the same as input case.
• Non-English or special characters in input are same in output.
• Non-English or special characters in password functions as "a" or "0 (zero)".
